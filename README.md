#sms77io Zammad Plugin

##Installation
1. Go to <b>Manage->Packages</b> and press <b>Choose File</b>
2. Choose <b>sms77.rb</b> and click <b>Install Package</b>
3. Go to <b>Manage->SMS->SMS Notification</b> and choose <b>sms77</b>
